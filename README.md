# Management-Karyawan
Management Karyawan adalah aplikasi contoh yang sangat sederhana untuk pembelajaran pemula dalam satu file PHP, poin pembelajaran CRUD dengan PHP, MySQLi procedural, Penanganan SQL Injection dengan mysqli prepared statements.

Untuk coba menjalankan aplikasi silahkan download dan buka file index.php isi `host`,`user`,`pass` sesuai dengan pengaturan database anda.
*Tidak perlu membuat database baru sendiri, aplikasi akan membuat database baru dengan nama "kantor" dan tabel "karyawan" secara otomatis.*
```
$host = 'localhost';
$user = 'root';
$pass = '';
```
Untuk pembelajaran silahkan ATMB(Amati,Tiru,Modifikasi,Buat sendiri).
